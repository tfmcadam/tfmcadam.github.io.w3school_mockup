Padding

Padding is Used to create space around an element's content, inside of any defined borders

As with Margin you have complete control over the padding. There are properties for setting the padding for each side of an element(top, right, bottom, and left).

Example:

This element has a padding of 70px

Try it yourself

Padding - Individual Sides

CSS has properties for specifying the padding for each side of an element:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

All the padding can have the following values:

  • length - specifies the padding in px, pt, cm, etc
  • % - specifies a padding in % of the width containing the element
  • inherit - specifies that the padding should be inherited from the parent element.

Note: Negative values are not allowed

Example

Set different padding for all four sides of an element


                    div {
                        padding-top: 50px;
                        padding-right: 30px;
                        padding-bottom: 50px;
                        padding-left: 80px;
                    }

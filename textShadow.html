Text Shadow

The text-shadow property adds shadow to text

In the simplest use, you only specify the horizontal shadow (2px) and the vertical shadow (2px):

Text Shadow Effect

Example


                h1 {
                    text-shadow: 2px 2px;
                }
Try it yourself

Next, add a color (red) to the shadow:

Text Shadow Effect!

Example


                h1 {
                    text-shadow:2px 2px red
                }
Try it yourself

More Text Shadow Examples

Text Shadow on a White Text

Example 1


                h1 {
                    color: white;
                    text-shadow: 2px 2px 4px #000000;
                }
